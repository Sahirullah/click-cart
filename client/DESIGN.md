# TrendTrunk - Professional Ecommerce UI Design

## Overview
A modern, professional ecommerce website UI built with React and CSS. The design focuses on clean aesthetics, smooth interactions, and excellent user experience.

## Design Features

### Color Scheme
- **Primary Color**: #ff9500 (Orange) - Used for highlights, CTAs, and hover states
- **Secondary Color**: #000 (Black) - Used for text and buttons
- **Background**: #ffffff (White) with light gray accents (#f8f8f8)
- **Text**: Dark gray (#1a1a1a) for readability

### Key Components

#### 1. Header
- Sticky navigation bar with logo and menu
- Search, wishlist, and shopping cart icons
- Cart badge showing item count
- Smooth hover animations on navigation links

#### 2. Hero Section
- Large, bold typography (3.5rem heading)
- Animated yellow shape background
- Featured product showcase with drop shadow
- Call-to-action button with hover effects
- Responsive grid layout

#### 3. Product Grid
- 8 featured products in a responsive grid
- Product cards with hover effects
- Quick view overlay on hover
- Add to cart functionality
- Product information (name, colors, price)

#### 4. Product Cards
- Image container with aspect ratio preservation
- Smooth image zoom on hover
- Quick view button overlay
- Add to cart button with icon
- Product details section

#### 5. Footer
- Dark background with white text
- Multiple sections (About, Customer Service, Legal, Social)
- Responsive grid layout
- Social media links

## Responsive Design

### Breakpoints
- **Desktop**: Full layout with 2-column hero
- **Tablet (768px)**: Adjusted spacing and font sizes
- **Mobile (480px)**: Single column layout, optimized touch targets

### Mobile Optimizations
- Flexible navigation menu
- Responsive product grid (2-3 columns)
- Optimized font sizes for readability
- Touch-friendly button sizes

## Animations & Interactions

### Hover Effects
- Navigation links: Underline animation
- Buttons: Color change and lift effect
- Product cards: Lift and shadow effect
- Product images: Subtle zoom effect

### Keyframe Animations
- **Float**: Hero shape floating animation (3s loop)
- **Pulse**: Hero badge pulsing animation (2s loop)
- **FadeIn**: Product overlay fade-in effect

## Typography
- **Font Family**: System fonts (-apple-system, BlinkMacSystemFont, Segoe UI, Roboto)
- **Headings**: Bold (700-800 weight)
- **Body**: Regular (400-500 weight)
- **Sizes**: Responsive scaling from 0.75rem to 3.5rem

## Spacing & Layout
- **Max Width**: 1400px container
- **Padding**: 2rem on desktop, 1rem on mobile
- **Gap**: 2rem between sections, 1.5rem between products
- **Border Radius**: 4-8px for cards and buttons

## Accessibility Features
- Semantic HTML structure
- Proper heading hierarchy
- Icon buttons with title attributes
- Focus states on interactive elements
- High contrast colors for readability
- Smooth scroll behavior

## Performance Considerations
- CSS Grid for efficient layouts
- Hardware-accelerated animations (transform, opacity)
- Optimized media queries
- Minimal repaints and reflows

## Browser Support
- Modern browsers (Chrome, Firefox, Safari, Edge)
- CSS Grid and Flexbox support required
- CSS custom properties (variables) support required
